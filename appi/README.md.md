# Appie 📱

> A modern app landing page with light/dark mode toggle, built with HTML, CSS, and JavaScript.

![Project Preview](project2-appie.png)

## 🔗 Live Demo
[View Project](https://chimerical-taiyaki-8e6b1a.netlify.app/)

## 🛠️ Built With
- HTML5
- CSS3
- JavaScript

## ✨ Features
- Light/Dark mode toggle
- Fully responsive design (mobile, tablet, desktop)
- Pricing tables section
- Step-by-step process visualization
- Smooth scroll animations
- Mobile-friendly navigation
- Testimonials section
- App store download buttons

## 📂 Project Structure

